Вот подробный пошаговый туториал для запуска вашего проекта «Тамагочи» на локальном компьютере. Веб-приложение состоит из бэкенда на Python (FastAPI) и фронтенда на чистом HTML/JS, поэтому процесс развертывания очень простой и не требует сложных инструментов.

---

## Шаг 1. Подготовка папок и файлов

Для начала создайте структуру проекта, чтобы сервер мог правильно находить файлы.

1. Создайте на компьютере новую папку, например, `tamagotchi-app`.
2. Внутри этой папки сохраните ваши файлы под точными именами:
* **`main.py`** (код бэкенда Python)
* **`index.html`** (код интерфейса)


3. Создайте внутри `tamagotchi-app` подпапку с именем **`assets`** (туда в будущем можно положить картинки персонажа: `happy.png`, `neutral.png`, `sad.png` и фон `room.png`, хотя даже без них интерфейс запустится и будет работать).

---

## Шаг 2. Установка Python и необходимых библиотек

Серверная часть написана на веб-фреймворке FastAPI. Для его работы нужен Python и несколько пакетов.

1. **Установите Python** (если еще не установлен):
* Скачайте Python версии 3.9 или новее с официального сайта [python.org](https://www.python.org/).
* **Важно:** При установке на Windows обязательно поставьте галочку **"Add Python to PATH"** (Добавить Python в переменную среды).


2. **Откройте терминал / командную строку:**
* **Windows:** Нажмите `Win + R`, введите `cmd` и нажмите Enter.
* **macOS / Linux:** Откройте приложение «Терминал».


3. **Перейдите в папку с проектом:**
Используйте команду `cd` и путь к вашей папке. Например:
```bash
cd C:\Users\ВашеИмя\Desktop\tamagotchi-app

```


4. **Установите зависимости:**
Введите команду для установки необходимых библиотек (`fastapi` для логики сервера и `uvicorn` для его запуска):
```bash
pip install fastapi uvicorn

```



---

## Шаг 3. Запуск сервера (Бэкенд)

Теперь, когда все готово, запускаем Python-сервер, который создаст базу данных и будет обрабатывать запросы от тамагочи.

1. Находясь в терминале в папке проекта, выполните команду:
```bash
uvicorn main:app --reload --port 8000

```


* *`main:app`* означает: искать объект `app` в файле `main.py`.
* *`--reload`* автоматически перезапускает сервер при изменении кода (удобно для разработки).
* *`--port 8000`* закрепляет сервер за портом 8000.


2. Если всё сделано правильно, в терминале появятся логи, а в конце будет написано:
`INFO: Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)`
3. В папке проекта автоматически появится файл базы данных **`tamagotchi.db`**.

> ⚠️ **Не закрывайте это окно терминала!** Если закрыть терминал, сервер перестанет работать.

---

## Шаг 4. Настройка и запуск интерфейса (Фронтенд)

Поскольку бэкенд и фронтенд запускаются раздельно, нужно убедиться, что HTML-страница знает, куда отправлять запросы.

1. Откройте файл `index.html` в любом текстовом редакторе (Блокнот, VS Code и т.д.).
2. Найдите в начале скрипта строку с переменной `API_URL`. Убедитесь, что она указывает на ваш локальный сервер:
```javascript
const API_URL = "http://127.0.0.1:8000";

```


3. **Запустите интерфейс:**
* **Вариант А (Простой):** Просто дважды кликните по файлу `index.html`. Он откроется в браузере как локальный файл (`file:///...`). Большинство функций (кнопки, анимации, отправка ЭЭГ) сразу начнут работать.
* **Вариант Б (Правильный):** Чтобы избежать возможных проблем с путями к картинкам (`assets/`), откройте `index.html` через локальный веб-сервер. Если вы используете редактор **VS Code**, просто установите расширение **Live Server** и нажмите кнопку *Go Live* в правом нижнем углу.



---

## Шаг 5. Проверка работы

1. Откройте приложение в браузере.
2. Попробуйте нажать на любую кнопку (например, **Питание** -> **Выпить стакан воды**).
3. Если полоса здоровья питомца обновилась, а в терминале Python пробежали строки со статусом `200 OK` — поздравляем, ваше приложение полностью работает локально!
4. Чтобы протестировать скрытую админ-панель: зажмите левую кнопку мыши на персонаже (или удерживайте палец, если тестируете с телефона) на **1.5 секунды** — на экране появится меню ручной настройки ХП, баллов и анимаций.
